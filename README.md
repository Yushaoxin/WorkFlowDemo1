# WorkFlowDemo1 简介

## 编译环境

- **开发工具**：Visual Studio 2022
- **目标框架**：.NET Framework 4.8
- **主要语言**：C#
- **依赖库**：System.Activities（工作流相关）

## 主要功能

WorkFlowDemo1 是一个基于 Windows Workflow Foundation (WF) 的工作流演示项目。主要功能包括：

- 通过自定义活动（Activity）实现文件的创建、内容写入和保存。
- 使用工作流（XAML）串联多个步骤，自动化文件操作流程。
- 支持输入文件名和内容，自动生成完整路径并保存文件。

主要流程如下：

1. **创建文件**：根据输入的文件名生成完整路径。
2. **输入内容**：接收并处理文件内容。
3. **保存文件**：将内容写入指定文件。

## 如何使用 Demo

1. **编译项目**  
   使用 Visual Studio 2022 打开解决方案，选择“生成”菜单进行编译。

2. **运行 Demo**  
   直接按 F5 或点击“启动调试”运行程序。

3. **演示流程**  
   程序入口在 `Program.cs`，会自动执行如下流程：
   - 传入文件名（如 `test.txt`）和内容（如 `这是一个测试文件的内容。`）。
   - 工作流自动依次执行创建文件、写入内容、保存文件等步骤。
   - 运行结束后，可在程序根目录下找到生成的文件。

4. **自定义参数**  
   如需更改文件名或内容，可修改 `Program.cs` 中 `WorkflowInvoker.Invoke` 的参数。
